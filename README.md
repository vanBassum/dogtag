# 🐾 DogTag

A simple, mobile-friendly page to help people return your lost items.

Attach a QR code to your belongings → finder scans → instantly contacts you.

## 🚀 Create your tag

👉 https://vanbassum.github.io/dogtag/edit

Fill in your details and copy the generated link.

## 📱 How it works

1. Open the editor  
2. Enter your contact details  
3. Copy the generated URL  
4. Turn it into a QR code  
5. Attach it to your item  

When someone scans it, they get:
- 📧 Email button  
- 📞 Call button  
- 💬 WhatsApp button  

All optimized for mobile → minimal effort to contact you.
